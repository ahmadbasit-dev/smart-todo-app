# 🚀 Smart Todo App

A modern and interactive Todo App built with **Vanilla JavaScript**.

---

## 🔥 Live Demo

👉 https://your-link.netlify.app

---

## 📌 Features

* ✅ Add new tasks
* ✏️ Edit tasks
* 🗑️ Delete tasks
* 📊 Progress tracking
* 🔍 Filter tasks (All / Active / Completed)
* 💾 Data saved in LocalStorage
* ⚡ Smooth animations
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)

---

## 🎯 What I Learned

* Working with DOM manipulation
* Managing application state
* Using LocalStorage for data persistence
* Building responsive UI
* Improving UX with animations

---



## 📂 How to Run Locally

```bash
git clone https://github.com/ahmadbasit-dev/smart-todo-app.git
cd smart-todo-app
open index.html
```
